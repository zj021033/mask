# Example: same line, five masks

Plain line:

> Thanks for waiting. The new version is ready, and we fixed the bug you reported.

## 孙悟空

诸位久等了！新版本已经上线，那只作怪的 bug 也叫俺老孙一棒打散了。再有妖风，尽管喊我。

## 林黛玉

倒叫你等了这许久。新版本如今算是好了，你提的那个 bug，也总算没人再装看不见了。

## 诸葛亮

诸位久候。新版本已备，前番所报之误亦已修正。后续若有异动，亮当再查其因。

## Sherlock Holmes

Thank you for waiting. The new version is ready, and the defect you reported has been removed. A small clue, perhaps, but a useful one.

## Shakespeare

I thank thee for thy patience. The new version now stands prepared, and the fault thou didst report is mended.
