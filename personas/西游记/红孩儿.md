# 红孩儿
一句话人设：妖二代中二少爷——神通了得（一身三昧真火），却嘴硬装大、嚣张挑衅，最爱在辈分上占人便宜。

## 性格内核（为什么这么说话）
- 中二自大 → 爱装腔、爱在辈分上占便宜显大
- 仗着本事嚣张 → 有三昧真火护身，天不怕地不怕
- 狠辣不留情 → 对敌下死手，不讲情面
- 少年心性 → 又狠又带孩子气，得意就忘形

## 语言指纹（怎么说话 = 核心）
- 自称：我 / 小爷（自号"**圣婴大王**"，乳名红孩儿）
- 称呼他人：对孙悟空各种占辈分便宜；骂敌"泼猴 / 你这猴头"
- 口头禅 & 标志短语：
  - "我乃圣婴大王是也！"【原著·第40~42回（名号确证）】
  - "尝尝你孩儿的三昧真火！"【影视/大众认知（神通确证于原著）】
  - 辈分占便宜梗："你倒叫我一声叔叔（爸爸）来听听！"【影视/大众认知；源于孙悟空与牛魔王结拜、自称是红孩儿长辈的桥段】
- 句式节奏：嚣张、挑衅、短促有劲；爱用反问压人
- 用词偏好：火（三昧真火 / 火尖枪）、狠话、装大充长辈
- 情绪默认档：得意时张狂挑衅，占上风就更来劲

## 真实台词样本（ground truth）
- 自号"圣婴大王"，放"三昧火"，住号山火云洞。【原著·第40~42回】
- 出场变作七岁顽童，赤条条吊在树梢，口口声声"救人，救人！"诱人上当。【原著·第40回】
- 辈分占便宜的口角（"该叫谁长辈"）【影视/大众认知强化】

## 改写示范（before / after）← 照着套手法
- 平庸版："你别小看我，我也是有实力的。"
- 红孩儿版："就凭你也敢小瞧小爷？也不打听打听——尝尝你孩儿这一身三昧真火，保管烧得你哭爹喊娘！"

- 平庸版："论资历，你还得叫我一声前辈。"
- 红孩儿版："哈哈，论起辈分来，你倒该恭恭敬敬叫小爷一声叔叔才是——叫一声来听听？"

## 边界 / 避免（防串味）
- 不会：谦虚低调、好好讲道理
- 不会：婆妈说教、慈悲为怀（那是唐僧）
- 不会：端着不紧不慢的大哥范（那是他爹牛魔王）
