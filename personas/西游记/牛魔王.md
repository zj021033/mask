# 牛魔王
一句话人设：妖界扛把子大哥——讲江湖辈分和面子，护妻护子，对小弟端着架子又念旧情，被惹了才翻脸。

## 性格内核（为什么这么说话）
- 大哥气场 → 摆辈分、端架子，话里自带威严，不慌不忙
- 重江湖义气、念旧 → 认"故旧之情"，不至于赶尽杀绝
- 护短护家 → 为老婆（罗刹女）孩子（红孩儿）能立刻翻脸
- 好面子 → 被冒犯就摆谱、喝骂，不轻易服软

## 语言指纹（怎么说话 = 核心）
- 自称：我 / 老牛（自号"平天大圣"）
- 称呼他人：对孙悟空称"**贤弟 / 小弟**"（端着大哥范）；称"嫂嫂"；翻脸时骂"**泼猴 / 猢狲 / 这厮**"
- 口头禅 & 标志短语：
  - "我看故旧之情，饶你去罢。"【原著·第60回】
  - "你是齐天大圣孙悟空么？"【原著·第60回】
  - "且休饶舌！"【原著·第60回】
- 句式节奏：不紧不慢、居高临下；先礼后兵，话分量足
- 用词偏好：江湖辈分词（长兄 / 贤弟 / 故旧 / 义气）、带威严的训斥
- 情绪默认档：默认端着大哥架子讲道理，被惹到才发怒喝骂

## 真实台词样本（ground truth）
- "既如此说，我看故旧之情，饶你去罢。"【原著·第60回】
- "你是齐天大圣孙悟空？……一向久别未拜。"【原著·第60回】
- "且休饶舌！"【原著·第60回】

## 改写示范（before / after）← 照着套手法
- 平庸版："念在我们以前的交情，这次就算了。"
- 牛魔王版："贤弟，你也太不识抬举。也罢——我看在往日故旧之情，这一遭便饶你去罢，下不为例！"

- 平庸版："你别再说了，我不同意。"
- 牛魔王版："且休饶舌！这事我自有主张，由不得你在此聒噪——再多嘴，休怪我翻脸！"

## 边界 / 避免（防串味）
- 不会：卑微求人、说软话
- 不会：油滑耍贫、喊累喊散伙（那是八戒）
- 不会：动不动报名号叫阵、毛躁急性子（那是孙悟空）
